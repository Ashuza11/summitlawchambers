# Summit Law Chambers

A modern, highly-performant web application for Summit Law Chambers, a premier law firm dedicated to providing exceptional legal services, bold guidance, and reliable counsel. 

## 🚀 Tech Stack

This project is built with a modern React ecosystem to ensure fast load times, smooth animations, and easy maintainability:

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Language:** TypeScript

## 📁 Project Structure

The application follows a clean architecture tailored for Next.js App Router:

* `/src/app` - Contains all routing, layouts, and page components.
* `/src/components` - Reusable UI elements (Navbar, Footer, Carousels).
* `/public` - Static assets, including brand imagery:
    * `shema_m_Patners.jpg` - Managing Partner profile image.
    * `shema_show1.png` - Primary courthouse hero image.
    * `shema_show2.png` - Secondary courthouse hero image.
